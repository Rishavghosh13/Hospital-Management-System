# Hospital Management System

## Overview

The **Hospital Management System** is a console-based Java application that facilitates the management of hospital operations, including handling patient and doctor information and scheduling appointments.
This project aims to streamline the workflow of hospital management and provide a user-friendly interface for interacting with the system.

## Features

- **View Doctor Details:**  
  Retrieve and display information about doctors including their ID, name, and specialization.

- **Add and View Patients:**  
  Add new patients to the system and view the details of existing patients.

- **Book Appointments:**  
  Schedule appointments between patients and doctors by providing patient ID, doctor ID, and the desired date.

- **Check Doctor Availability:**  
  Ensure that doctors are available on the selected date before booking an appointment.

## Technologies Used

- **Java:** Core language for implementing the logic and functionality of the application.
- **JDBC (Java Database Connectivity):** For connecting and interacting with the MySQL database.
- **MySQL:** The relational database management system used to store data about patients, doctors, and appointments.
- **Properties File:** Used for storing database configuration details.
- **Git:** Version control system for managing and tracking changes to the project.
